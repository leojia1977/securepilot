# 任务：修复代理问题导致的连接超时

## 背景
系统设置了 HTTP_PROXY，导致所有 httpx/requests 请求
包括 localhost 都被代理拦截，造成超时。

## 修复一：tests/sprint3_acceptance.py
在文件顶部 import 之后，加入以下代码：

import os
os.environ["NO_PROXY"] = "localhost,127.0.0.1"
os.environ["no_proxy"] = "localhost,127.0.0.1"

## 修复二：tests/sprint1_acceptance.py 和 tests/sprint2_acceptance.py
同样在文件顶部加入相同的两行代码。

## 修复三：src/api/main.py
在文件顶部加入：
import os
os.environ["NO_PROXY"] = "localhost,127.0.0.1"
os.environ["no_proxy"] = "localhost,127.0.0.1"

## 修复四：mock/vllm_server.py
同样在文件顶部加入相同两行。

## 完成后说"TASK完成"
